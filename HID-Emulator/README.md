# HID-Emulator

Projects is part of [Digispark HID-Emulator](https://github.com/aenniw/ATTINY85/tree/master/HID-Emulator) project.

#### Used pin-outs

| Wemos D1  | ESP-8266 | Led Strip | Function                      |
|:---------:|:--------:|:---------:|:-----------------------------:|
| TX        | TXD      | 1         | TXD                           |
| RX        | RXD      | 3         | RXD                           |
| G         | GND      | -         | Ground                        |
| 5V / 3V3  | - / 3.3V | -         | Power                         |
