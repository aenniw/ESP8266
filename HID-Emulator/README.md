# HID-Emulator

Projects is part of [Ambi-Light](https://github.com/aenniw/SamyGo/tree/master/Ambi-Light) project.

#### Used pin-outs

| Wemos D1  | ESP-8266 | Led Strip | Function                      |
|:---------:|:--------:|:---------:|:-----------------------------:|
| RX        | RXD      | DATA      | RXD                           |
| G         | GND      | GND       | Ground                        |
| 5V / 3V3  | - / 3.3V | +V        | Power                         |
